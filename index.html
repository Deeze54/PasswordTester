This page is just the interface — it needs a cobalt server to do the actual downloading. The public one is blocked for YouTube, so run your own (free, ~2 minutes):
  1. Open the Railway one-click template and deploy. It presets CORS_WILDCARD=1 so this page can reach it.
  2. Copy the public URL Railway gives you (ends in .up.railway.app).
  3. Paste it above and hit Save. Done — it's remembered on this device.
Prefer another route? Any cobalt instance with permissive CORS works. Public instances are hit-or-miss for YouTube.
